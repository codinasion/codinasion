## @codinasion/code

A collection of useful TypeScript code snippets.

### Installation

```bash
# if you're using pnpm
pnpm add @codinasion/code

# or, if you're using npm
npm install @codinasion/code

# or, if you're using yarn
yarn add @codinasion/code
```
