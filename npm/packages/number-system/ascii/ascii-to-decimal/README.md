# `@codinasion/ascii-to-decimal`

## Installation

```bash
# if you're using pnpm
pnpm add @codinasion/ascii-to-decimal

# or, if you're using npm
npm install @codinasion/ascii-to-decimal

# or, if you're using yarn
yarn add @codinasion/ascii-to-decimal
```

<div align="center">
  <img src="https://raw.githubusercontent.com/codinasion/codinasion/master/assets/rainbow-hr.png" alt="rainbow hr" width="100%" height="70%">

## Usage

```javascript
import { AsciiToDecimal } from "@codinasion/ascii-to-decimal";

const ascii = "Hello World";

const decimal = AsciiToDecimal(ascii);
console.log(decimal);

// Output
// [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]
```
