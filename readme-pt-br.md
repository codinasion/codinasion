Contribuindo
Veja [o guia de contribuição][9] para instruções detalhadas sobre como começar a contribuir para o nosso projeto.

Aceitamos [diferentes tipos de contribuições][12], incluindo algumas que não requerem que você escreva uma única linha de código.

Se você está procurando uma maneira de contribuir, você pode vasculhar nossos [problemas existentes][10] para encontrar algo para trabalhar. Quando estiver pronto, confira [Como começar a contribuir][9] para instruções detalhadas.

<div align="center">
  <img src="https://raw.githubusercontent.com/codinasion/.github/master/assets/rainbow-hr.png" alt="linha de arco-íris" width="100%" height="70%">
</div>
Junte-se a nós em discussões
Usamos as Discussões do GitHub para conversar sobre todos os tipos de tópicos relacionados à documentação e a este site. Por exemplo: se você precisa de ajuda para solucionar um PR, tem uma ótima nova ideia ou deseja compartilhar algo incrível, junte-se a nós nas [discussões][11].
